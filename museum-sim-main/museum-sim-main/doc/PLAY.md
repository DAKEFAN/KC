# 游玩说明

## 1. 基本流程

游戏以“季度”为单位推进。

每个季度通常会经历以下循环：

1. 处理右侧的行政任务
2. 在办公室推进展览、申报科研，或去大学/家里安排个人成长
3. 根据手头状态决定是否摸鱼回血或花钱补状态
4. 点击“结束本季度”进入结算

季度结算时会发生：

- 馆内经费到账
- 工资发放到个人存款
- 随机事件触发
- 年度切换时年龄增加、大学进度更新

## 2. 核心资源

- 精力：很多工作都会消耗；归零后风险极高
- 愉悦：代表精神状态；长期过低会触发坏结局
- 公款：用于展览相关支出，不能当成个人钱
- 存款：个人资金，用于消费、进修和解锁家庭
- 智商 / 情商 / 声望：影响升职、进修和部分系统解锁

## 3. 行政任务

右侧“行政任务”每季度都要处理一次。

- 点击“尝试甩锅 (Roll)”会掷骰子
- 点数 `>= 3`：甩锅成功，本季度可继续推进展览
- 点数 `< 3`：甩锅失败，展览面板会被锁定一个季度

如果连续几次都先做展览、后回领导消息，还有概率触发额外惩罚。

## 4. 展览系统

展览是办公室里的核心玩法。

- 先申请新展览
- 再依次推进资料收集、阅读整理、调研、主题、展品、设计等任务
- 每项任务都要消耗经费，也会触发对应剧情事件
- 展览完成后需要等待反馈报告，再结算结果

建议优先保证：

- 本季度行政任务已处理
- 公款足够
- 精力不要被提前榨干

## 5. 科研系统

- 每年 `Q1` 可以申报科研项目
- 每年 `Q4` 统一结算结果
- 通过后会获得大量经费和声望

科研窗口是很重要的中期成长来源，尽量不要错过。

## 6. 大学与学历

大学界面分为两种玩法：

- 短期进修：花钱上课，快速提升属性
- 在读硕士 / 博士：按学期选课、修学分、写论文、申请毕业

注意点：

- 申请入学只在每年 `Q2`
- 申请在读项目需要智商、情商和存款达到门槛
- 学位项目有修读年限，拖太久会延毕
- 博士毕业后会显著提高智商上限和综合能力

## 7. 家庭系统

- 当个人存款达到 `100,000` 元时解锁
- 解锁后，每个季度可以回家休息一次
- 回家休息会稳定恢复精力和愉悦

这是比摸鱼更稳定的恢复手段，但解锁时间通常会晚一些。

## 8. 晋升与结局

职称评定入口在左侧人物卡中，会在合适季度开放。

想要稳定升职，通常需要同时兼顾：

- 工龄
- 智商 / 情商成长
- 声望积累
- 身心状态

游戏中存在多个结局，包括但不限于：

- 馆长结局
- 光荣退休
- 劳碌命
- 不如回家
- 庸碌一生

## 9. 自动存档

- 游戏会把当前进度保存到浏览器本地
- 刷新页面后，开始页会自动显示“继续游戏”
- 设置菜单中的“重置所有数据”会清空本地存档

## 10. 开局建议

- 先处理行政任务，再决定要不要碰展览
- 精力低时优先商店或摸鱼，别硬撑到归零
- 每年记住两个关键时间点：`Q1` 科研申报，`Q2` 学位申请
- 存款不要全花光，后续进修和家庭系统都要用
- 展览不是越多越好，先把一个项目做顺更重要

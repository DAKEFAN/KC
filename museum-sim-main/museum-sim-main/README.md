# 博物馆生存指南

一个纯前端的网页小游戏。你将扮演刚入行的博物馆从业者，在季度循环里平衡行政杂活、展览策划、科研申报、学历进修和个人状态，目标是在职业消耗中活下来，并尽量走到更高职位。

## 快速开始

1. 克隆仓库

```bash
git clone https://github.com/yuexin-UvU/museum-sim.git
cd museum-sim
```

2. 直接打开 `index.html`

- 最简单的方式：双击 `index.html`
- 如果你更习惯本地服务器，也可以执行 `python -m http.server` 后在浏览器访问

## 当前版本特性

- 纯静态页面，无需安装依赖、无需构建
- 本地自动存档，刷新页面后可继续游戏
- 单局核心循环完整：行政任务、展览推进、科研申报、大学系统、职称晋升、结局判定
- 家庭场景已可解锁，并提供每季度一次的恢复型操作

## 主要文件

- `index.html`：页面结构与主要交互入口
- `styles.css`：整体视觉样式与场景布局
- `main.js`：页面初始化与轻量交互
- `game.js`：游戏状态、系统逻辑与 UI 更新
- `data.js`：文本数据库、数值配置和事件池
- `doc/PLAY.md`：完整游玩说明
- `doc/DEBUG.md`：开发期调试记录

## 游玩文档

- 玩法说明：[doc/PLAY.md](doc/PLAY.md)
- 掷骰子演示页：[doc/dice.html](doc/dice.html)

## 说明

- 自动存档保存在浏览器 `localStorage`
- 点击游戏内左上角设置菜单的“重置所有数据”会清空本地存档并重新开始
- 这是一个单机项目，当前没有后端同步或账号系统
